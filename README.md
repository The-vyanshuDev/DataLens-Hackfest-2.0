An intelligent data dictionary platform that automatically connects to enterprise databases, extracts and profiles schema data, and generates business-friendly documentation. 
It combines structured processing with AI-powered exploration to make data understandable, searchable, and actionable.

Key Features :
- Auto-connects to PostgreSQL, Mysql, SQL Server
- Deterministic schema extraction & quality profiling
- Gemini-powered documentation generation
- Google ADK agent for natural language querying & SQL generation
- Data can be exported into json and markdown format.
- Chat interface added for a faster response from the agent and a better clarity on schema. 
- Provides a graphical representation of the data and its inter-relational dependencies present inside the database.
